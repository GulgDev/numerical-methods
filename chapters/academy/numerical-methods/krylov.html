    Алексей Крылов

    Алексей Николаевич Крылов — выдающийся русский и советский учёный-математик, механик и инженер-кораблестроитель.

    Алексей Крылов родился 15 августа 1863 года в селе Висяга Симбирской губернии в семье артиллерийского офицера Николая Александровича Крылова. По традиции, Алексея Николаевича ждала судьба военного, но большее влияние на него оказало окружение многочисленных родственников, которые в дальнейшем стали знаменитыми русскими врачами, учёными, композиторами.

    В декабре 1914 года он был избран членом-корреспондентом Академии наук, а в апреле 1916 года — ординарным академиком по математической физике.

    Метод Крылова

    Рассмотрим пружинный маятник и представим его движение в матричном виде, введя вектор состояния y = ( v x ) : dy dt = ( 0 1 - k m - c m ) ⁢ y Найдём частоту колебания маятника с помощью метода Крылова.

    Стоит также отметить, что эту и подобные ей задачи (вычсиление коэффициентов характеристического полинома) можно решить и “традиционным” способом, но для больших матриц нахождение корней получающихся уравнений становится очень сложным. Вышеупомянутый же метод сводится к простому умножению матрицы на вектор и решению системы линейных уравнений.

    Пусть B = ( 0 1 - k m - c m ) , тогда: dy dt = B ⁢ y

    1. Выберем случайный ненулевой вектор b0 = ( 1 0 ) .
    2. Будем последовательно умножать матрицу B размером n×n на этот вектор, получая серию векторов b1 , b2 , …, bn : b1 = B ⁢ b0 = ( 1 - k m ) b2 = B ⁢ b1 = ( - k m - k m + c ⁢ k m 2 )
    3. Найдём коэффициенты c0 , c1 , …, c n - 1 : bn + c n - 1 ⁢ b n - 1 + … + c1 ⁢ b1 + c0 ⁢ b0 = 0
    4. Рассмотрим характеристический полином: P ( λ ) = λn + c n - 1 ⁢ λ n - 1 + … + c1 ⁢ λ + c0 Это многочлен, определяющий в данном случае поведение нашего маятника. Его корни называются собственными значениями. Подставим в него найденные коэффициенты и решим уравнение P ( λ ) = 0 .

    Мы получили два комплексно-сопряженных собственных значения α ± β ⁢ i (пара комплексных чисел, у которых действительные части равны, а мнимые части противоположны по знаку). Действительная часть α характеризует амплитуду колебаний ( A = e α ⁢ t ), мнимая часть β равна частоте колебаний в радианах в секунду.
